Module operations that can be performed in [[notes/Mathematics/Discrete Mathematics/Number Theory/Modular Arithmetics]].

## Addition

$$
(A + B) \bmod C = (A \bmod C + B \bmod C) \bmod C
$$

## Subtraction

$$
(A - B) \bmod C = (A \bmod C - B \bmod C) \bmod C
$$

## Multiplication

$$
(A * B ) \bmod C = (A \bmod C * B \bmod C) \bmod C
$$
